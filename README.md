# Login React App

### Git Clone
```bash
git clone git@github.com:Gustiana882/react-login-26-10-2021.git
```

### Install Package
```bash
cd react-login-26-10-2021

yarn install
```

### Run App
```bash
yarn start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build App
```bash
yarn run build
```
